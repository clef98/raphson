PCA or principal component analysis is a optimaztion method that helps better visualize input data.
It is considered a dimensionality reduction algorithm. This means it can covert lets say a input data that is 5d
to mabye 2d or 3d so that you can better visualize it. In our example, we will be using a 3d input data which we
will want to convert to 2d. In theory, we want to take a 2d slice of our 3d data that is of best fit. We then
project the data on to the 2d plane which then we can use as our input.
