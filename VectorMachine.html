A support vector machine is a improves version of logistic regression. It uses a kernel
(usually gaussian kernel) to determine a "line" to seperate the data into classes hence
classification. This is where it diverges from a logistic regression model. Instead of
computing the vertical delta(y delta), it computes the closest distance which can often
lead to a better, more spacious seperation of data.
