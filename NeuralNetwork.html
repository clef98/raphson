A neural network is modeled and heavily based on neurons in your brain. Each neuron has a nucleus
and axons which connect each nucleus. Each nucleus can send an electric pulse of varying voltage.
With this knowledge, we can construct a neural network with nucelei represented by circles called nodes,
and axons represented by lines. Each line will be assigned a value of voltage it would
output. A simple representation would have 3 rows of nodes with the first row as the input and the last
row as the output. Neural networks are a form of logistic regression. They will output a value 0-1
depending on its "confidence"(ex. 0.6 indicate 60% certainty).
