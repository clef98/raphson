A neural network is modeled and heavily based on nuerons in your brain. Each neuron has a
nucleus and axons which connect each nucleus. Each nucleus can send an eletric pulse that can vary in volotage. Through this knowlegde, we can construct a neural network with axons
represented by circles, and axons represented by lines. Each line will be assigned a value which would be how much volatage it would output.
