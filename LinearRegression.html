In the most basic form, linear regression is the line of best fit. For example, if there was a table of points plotted on the coordinate plane, the line of best
fit would simply just be the line that seems most like the plotted points. Now, to mathematically find the best line, we would need to devolp a way to
measure how far off the line is from the points. This function is called the cost function. We also
need data points which can be labled with a x(i) and y(i) where x and y are the ith data point in
the data set. We also need to define the parameter which in this case(2D plane) are θ0 and θ1
where θ0 is equal to 1. This line can then be represented by:
$$h_\theta(x) = \theta_0 + \theta_1 x $$
The cost function can also be defined as:
$$J(\theta_0, \theta_1) = \frac{1}{2m} \sum_{i=1}^m (h_\theta(x^{(i)}-y^{(i)}))^2$$
Where m is the number of data points. The goal is to find the θ0 and θ1 that minimize the cost function. The equation for these θ's is defined as:
$$\theta_0 = \theta_0 - \alpha\frac{1}{m}\sum_{i=1}^m (\frac{\partial}{\partial\theta_0}J(\theta_0, \theta_1) = \theta_0 - \alpha\frac{1}{m}\sum_{i=1}^m (h_\theta(x^{(i)})-y^{(i)})$$ $$\theta_1 = \theta_1 - \alpha\frac{1}{m}\sum_{i=1}^m (\frac{\partial}{\partial\theta_1}J(\theta_0, \theta_1) = \theta_1 - \alpha\frac{1}{m}\sum_{i=1}^m (h_\theta(x^{(i)})-y^{(i)}) * x^{(i)}$$
Repeat untill they converge to a value. If they do not converge but grow, then decrease alpha's value. Once you find out the values of the θ's,
you can directly plug it in to the hypothesis and get your line. You can also scale up by using higher power of x or more parameters or both!
