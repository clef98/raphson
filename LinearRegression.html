In the most basic form, linear regression is the line of best fit. For example, if there was a table of points plotted on the coordinate plane, the line of best fit
would simply just be the line that seems most like the plotted points. Now, to mathematically find the best
line, we would need to devolp a way to measure how far off the line is from the points. This function is
called the cost function. We also need data points which can be labled with a x(i) and y(i) where x and y are
the ith data point in the data set. We also need to define the parameter which in this case(2D plane)
are θ0 and θ1 where θ0 is equal to 1. This line can then be represented by y or hθ = θ0 ⁺ θ1x. The cost function
can also be defined as:
J(θ0, θ1) = $\frac{1}{2m} \sum_{i=1}^m (h_\theta(x^{(i)}-y^{(i)}))^2$
