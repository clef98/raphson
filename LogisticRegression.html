Logistic regression models the probabilities for classification problems with two possible outcomes.
It's an extension of the linear regression model for classification problems. Instead of fitting a straight
line or hyperplane, the logistic regression model uses the logistic function to squeeze the output of
a linear equation between 0 and 1. The logistic function is defined as:
$$G(z) = \frac{1}{1+e^(-z)} $$
