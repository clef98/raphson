Logistic regression is a classification algorithm meaning that the function is to be able to
classify a input such as a schoolbus, truck, etc. This model in the basic form will output a number 0-1
depending on its certainty(if it outputs 0.7, it is 70% sure and vice versa). Similiar to linear regression,
we need to start out with a hypothesis to be able to seperate the input(shown to the right).
This hypothesis is defined as:
$$ z = \theta ^Tx $$ $$g(z) = \frac{1}{1+e^{-z}} $$ $$ h_\theta (x) = g(\theta ^Tx )$$
Where x is the input, theta is the parameters, and y is the output(will be used later).
Also like linear regression, we need a way to define how well the our hypothesis matches the output/y.
This equation is called the cost function which is defined as:
$$J(\theta) = -\frac{1}{m} \bigl(\sum_{i=1}^m y&{(i)}logh_\theta(x^{(i)})+(1-y^{(i)})log(1-h_\theta(x^{(i)}))\bigr)$$ Note: y = 0 or 1 always.
