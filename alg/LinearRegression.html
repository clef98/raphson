In its most basic form, linear regression is the line of best fit. For example, if there was a table of points plotted on the coordinate plane, the line of best
fit would simply be the line that most closely represents the plotted points. To mathematically find the best line, we need to develop a method to
measure the discrepancy between a sample line and data points. This function is the cost function. We also need data points which can be labeled
with an x(i) and y(i) where x and y are the ith data point in the data set. We need to define the parameter which in this particular 2D plane are θ0 and θ1
where θ0 is equal to 1. This line can then be represented by
$$h_\theta(x) = \theta_0 + \theta_1 x $$
The cost function can also be defined as
$$J(\theta_0, \theta_1) = \frac{1}{2m} \sum_{i=1}^m (h_\theta(x^{(i)}-y^{(i)}))^2$$
where m is the number of data points. The goal is to find the θ0 and θ1 that minimize the cost function. The equations for these θ's is defined as
$$\theta_0 = \theta_0 - \alpha\frac{1}{m}\sum_{i=1}^m (\frac{\partial}{\partial\theta_0}J(\theta_0, \theta_1) = \theta_0 - \alpha\frac{1}{m}\sum_{i=1}^m (h_\theta(x^{(i)})-y^{(i)})$$ $$\theta_1 = \theta_1 - \alpha\frac{1}{m}\sum_{i=1}^m (\frac{\partial}{\partial\theta_1}J(\theta_0, \theta_1) = \theta_1 - \alpha\frac{1}{m}\sum_{i=1}^m (h_\theta(x^{(i)})-y^{(i)}) * x^{(i)}$$
and are iterated until they converge to a value. If the theta values do not converge but grow, then decrease alpha's value. Once you find the values of the θs,
you can directly input them into the hypothesis and arrive at the line of best fit. You can also scale up by higher orders of x or more parameters or both!
This algorithm as well as Logistic Regression are the basic building blocks for all machine learning algorithms.
